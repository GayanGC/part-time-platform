⚡ Part Time - Student Job Marketplace

Part Time** is a dual-platform solution designed to connect university students with short-term employment opportunities. The system simplifies the hiring process for event organizers and employers while providing students with a professional way to find work and verify their attendance.

🚀 Key Features

 🖥️ Web Dashboard (For Job Posters)
- Job Posting:** Create detailed job listings with requirements like salary, location, and student capacity.
- Applicant Management:** Real-time tracking of student applications with Approve/Reject functionality.
- QR Attendance Scanner:** Integrated camera scanner to verify student attendance on-site instantly.
- Live Analytics:** Monitor active jobs and total applicants.

📱 Mobile App (For Students)
- Job Discovery:** Browse and search for part-time jobs filtered by University or Location.
- Instant Application:** One-tap application process with real-time status updates.
- Digital Attendance ID:** Automatic generation of a unique QR code upon job approval.
- Profile Persistence:** Secure login and session management.

🛠️ Tech Stack
- Frontend:** React.js (Vite), Tailwind CSS
- Mobile:** React Native (Expo)
- Backend/Database:** Firebase (Authentication, Cloud Firestore)
- Icons & UI:** Lucide React, Expo Vector Icons
